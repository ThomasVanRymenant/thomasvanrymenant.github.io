Uitleg - Vergelijk getelde inventaris met inventaris uit exactonline

  1. Selecteer 2 excel-files met getelde inventaris.

    Let op:

    De namen van de kollommen in beide files moeten op exact dezelfde manier geformatteerd zijn als op onderstaande afbeelding.

  1. Selecteer de excel-file met getelde inventaris als File 1.

    Let op:
    • De namen van de kollommen in file 1 moeten exact dezelfde manier geformatteerd zijn als op onderstaande afbeelding.
    • De file met getelde inventaris hoeft niet volledig ingevuld te zijn. Enkel voor artikelen met een genoteerde inventaris wordt gechecked of de genoteerde inventaris overeenkomt met de inventaris van datzelfde artikel in File 2.
    • Indien de inventaris van een artikel wordt geteld (in het magazijn), en het blijkt dat er 0 stuks zijn. Dient dat aangegeven te worden dmv een 0 in de kolom Tal1. Anders wordt deze inventaris niet gechecked door het programma.
  2. De benodigde File 2, met de inventaris van de artikelen volgens exactonline, moet uit exactonline geexporteerd worden.

    Stappen om de gepaste file te verkrijgen uit exactonline:

    1) Ga naar "Voorraad", "Artikelen", en klik "Overzicht" onder batchnummers

    2) Klik op "Artikel"

    3) Filter op beschikbaarheid: selecteer "Ja"

    4) Klik "Vernieuwen"

    5) En dan exporteren die handel
